
LINEARIZED LONGITUDINAL EQUATIONS OF MOTION FOR SAAB 2000 and 340


Acknowledgement:  The models were provided by Jon Bergstrom, SAAB AB.


Four cases are given for two different aircraft (Saab 2000 and Saab
340):

1, Approach, Flaps 20 degrees, gear down, Speed Vref (approach speed)
   trimmed at slope -3 degrees.

2, Cruise, Mach 0.5, H=20000ft

3, Max cruise, Max altitude and Mach number for each aircraft.

4, Go around, Max thrust, accelerated climb
   5 degrees for Saab 2000 and 3 degrees for Saab 340.
   Flaps 20, gear up.

Example session:
----------------

>> load S2000-3
>> Initlong2000         Set up matrices etc
>> eig(ALONG)           Gives the longitudinal stability eigenvalues
>> Linlong2000          Set up Simulink
>> Plotlong2000         Plot response

Autopilot:
----------

The Saab 2000 model has a simplified autopilot which may be
incorporated by adjusting the gain K in the block that you want to
activate. Set K to the general, speed compensated gain KE to activate
and adjust the resp.  PID gains.

The gains are appropriate for theta-control, but other may need to be
adjusted.  The reference signal can be given in the step-box by each
PID regulator. Note that the reference signal is a delta to the
trimmed value.
